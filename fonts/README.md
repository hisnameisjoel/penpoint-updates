# Penpoint CJK UI Font Packs

On-demand font packs downloaded by the Penpoint desktop app when a user switches
the interface language to Chinese (Simplified) or Japanese. Kept out of the app
installer to save ~46MB for all users; fetched here only when needed.

- `noto-sans-sc/` — Noto Sans SC (Simplified Chinese), weights 400/500/600/700
- `noto-sans-jp/` — Noto Sans JP (Japanese), weights 400/500/600/700

Source: Google Noto fonts via @fontsource (single-file per-weight woff2, full CJK range).
